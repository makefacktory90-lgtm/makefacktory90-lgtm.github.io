1. ПРЕЗЕНТАЦИИ
NotebookLM — Из документов в слайды
→ notebooklm.google.com
Когда: У вас есть PDF, Word, статья → нужна презентация
Роль: дизайнер презентаций McKinsey. Аудитория: руководители отделов маркетинга Для каждого слайда: — Заголовок = вывод. Не "Продажи Q4", а "Продажи Q4 выросли на 47% год к году" — Максимум 3 тезиса, каждый ≤ 12 слов — Каждый тезис = факт из источника с цитатой Первый слайд: самый удивительный инсайт из материала. Тот, ради которого стоит смотреть дальше. Предпоследний: сравнение "было → стало" Последний: одно действие. Не "Спасибо" Формат: Detailed Deck Стиль: минималистичный, много воздуха Язык: русский
Kimi — Исследование + презентация
→ kimi.ai
Когда: Нужны свежие данные + презентация (бесплатно)
Ты — аналитик-исследователь уровня McKinsey. Твоя задача: создать презентацию с глубоким исследованием, которая меняет представление аудитории о теме. Тема: [ВАША ТЕМА] Аудитория: [КТО СМОТРИТ] Режим: Visual КРИТИЧЕСКОЕ ПРАВИЛО: Прежде чем писать, ответь на три вопроса: 1. Какое общепринятое мнение я опровергну? 2. Какие данные 2024-2026 это докажут? 3. Что аудитория будет делать ИНАЧЕ после? СТРУКТУРА (11 слайдов): Слайд 1 — Провокация: "Все думают [общепринятое]. Данные 2024-2026 говорят [противоположное]." Одна цифра, которая ломает шаблон. Источник обязателен. Слайды 2-4 — Реальная картина: Три ключевых тренда с данными. Для каждого: заголовок-вывод + 2-3 факта с цифрами + источник + график Слайд 5 — Таблица сравнения: Кто делает правильно vs кто отстаёт. Минимум 4 компании. Цвета: зелёный = лидер, красный = отстающий Слайды 6-8 — Что это значит для вас: Три практических вывода. Формула: "Если вы [ситуация], то [конкретное действие]" Запрещено: "оптимизировать", "улучшить". Только конкретика. Слайды 9-10 — План действий: 3 шага по убыванию сложности, возрастанию импакта. Для каждого: что делать + сколько времени + какой результат (измеримый) Слайд 11 — Источники ПРАВИЛА: — Только данные 2024-2026 — Каждая цифра — со ссылкой — Один факт, который ПРОТИВОРЕЧИТ общепринятому мнению — Источники: Gartner, Forrester, McKinsey, Deloitte, HBR Слайдов: 11
Gamma — Быстро и красиво
→ gamma.app
Когда: Нужна презентация за 3 минуты
Создай структуру презентации для Gamma. Тема: [ТЕМА] Аудитория: [КТО] Цель: [ЧТО СДЕЛАЮТ ПОСЛЕ] 7 слайдов: - Слайд 1: Hook — вопрос без очевидного ответа ИЛИ число, которое удивит - Слайд 2: Проблема (The Problem) — цена бездействия в цифрах - Слайд 3: Решение (The Solution) — 3 тезиса, каждый ≤ 10 слов - Слайд 4: Как работает (Process) — максимум 3 шага - Слайд 5: Доказательство — один кейс или метрика - Слайд 6: Возражения — ответы на 2-3 главных "но" - Слайд 7: CTA — одно действие + дедлайн Для каждого слайда: — Заголовок (вывод, не тема) — Текст (≤ 40 слов) — Визуал (график/схема/иконка) ТЕСТ: прочитай ТОЛЬКО заголовки — история понятна? Язык: русский
2. ТЕКСТЫ
Claude — ToV-карта бренда
→ claude.ai
Когда: Нужно проанализировать стиль бренда
Ты профессиональный лингвист-стилист и эксперт по анализу авторского текста. Твоя задача: провести глубокий анализ предоставленных примеров текста и создать детальное руководство, которое позволит в точности воспроизвести авторский стиль в новых текстовых постах. Проанализируй и систематизируй следующие аспекты (все особенности пиши в одну строку после названия аспекта, разделяя точкой с запятой): Тональность и стиль: {Проведи детальный анализ эмоционального тона текста: формальный/неформальный, серьезный/юмористический, авторитетный/скромный. Определи преобладающие эмоциональные оттенки: вдохновляющий, провокационный, информативный, критический и т.д. Отметь используемые риторические приемы, метафоры, аналогии. Определи, как автор позиционирует себя: как эксперт, друг, наставник, исследователь.} Лексические особенности: {Проанализируй словарный запас: простой/сложный, обыденный/специализированный, современный/классический. Выяви характерные слова-маркеры, устойчивые выражения, авторские неологизмы, фразеологизмы. Определи наличие профессиональной терминологии, иностранных слов, сленга. Создай список из 10-15 наиболее характерных для автора слов и выражений.} Синтаксические особенности: {Определи типичную длину предложений, их структуру и сложность. Проанализируй соотношение простых и сложных предложений. Отметь особенности использования вводных конструкций, причастных и деепричастных оборотов. Выяви характерные синтаксические приемы: инверсия, параллелизм, эллипсис, риторические вопросы.} Структура контента: {Проанализируй композиционное построение текста: вступление, основную часть, заключение. Определи типичные приемы начала текста (хуки): вопрос, провокация, история, статистика. Выяви логические переходы между частями текста, наличие подзаголовков. Определи ключевые элементы завершения: вывод, призыв к действию, риторический вопрос. Опиши, как развивается основная мысль на протяжении текста.} Форматирование и визуальная подача: {Проанализируй длину и структуру абзацев, использование маркированных и нумерованных списков, цитат, выделений текста. Определи типичное количество предложений в абзаце. Отметь особенности использования заголовков и подзаголовков, их оформление. Проанализируй визуальное структурирование информации.} Орфография, пунктуация и типографика: {Отметь особенности использования знаков препинания: тире, двоеточия, скобки, многоточия. Проанализируй особенности оформления цитат и диалогов. Определи, обращается ли автор к читателю, и какую форму обращения использует (ты/вы). Выяви типографские предпочтения: использование определенного типа кавычек, тире, особенности оформления списков.} Стилистические приемы и фигуры речи: {Проанализируй использование метафор, сравнений, эпитетов, гипербол, иронии, аллитерации и других стилистических приемов. Выяви наличие историй, примеров из жизни, статистики, экспертных мнений. Определи баланс между эмоциональностью и фактологичностью.} Взаимодействие с аудиторией: {Проанализируй, как автор вовлекает читателя: прямые обращения, риторические вопросы, императивы. Определи степень диалогичности текста, создания эффекта личного общения. Выяви способы проявления эмпатии к читателю, предвосхищения возражений, создания доверия.} В ответ предоставь только результат анализа по указанным разделам, без вступительных объяснений или комментариев. → claude.ai

Telegram: Лаконичный пост с bullet points и вовлекающим вопросом 2. Email: Структурированное письмо с интригующим Subject Line и CTA 3. Stories: Серия из 5 карточек (Текст + Визуальное описание) 4.
