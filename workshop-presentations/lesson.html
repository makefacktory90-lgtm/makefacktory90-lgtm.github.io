Теория презентаций

Принципы, которые работают без любого AI

01
Заголовок = вывод
Не «Продажи Q4», а «Продажи Q4 выросли на 47% год к году». Тест: прочитай только заголовки — вся история понятна?
02
Один слайд = одна мысль
Максимум 3 буллита. Каждый ≤ 10 слов. Если на слайде больше — разбейте на два слайда.
03
Storyline: хук → доказательства → действие
Проблема (2-3 сл.) → Решение (3 сл.) → Доказательства (2 сл.) → Одно действие (1 сл.). Первый слайд — провокация, последний — дедлайн.
04
Данные > мнения
Числа > слова. Графики > описания. Одна убедительная метрика сильнее трёх абзацев текста.
Антиправила Запрещённые фразы: «в современном мире», «стоит отметить», «важно понимать». Первый слайд — никогда «Добро пожаловать». Последний — никогда «Спасибо» и «Вопросы?».

Промт-конструктор

Один промт → структура из любой темы

Главный промт воркшопа. Работает в любой нейронке: Claude, ChatGPT, Gemini, Kimi. Подставьте свою тему, аудиторию и цель — получите структуру презентации McKinsey-уровня за 30 секунд.

Алгоритм Тема → этот промт → структура слайдов с заголовками-выводами. Потом — в любой инструмент: Gamma, Kimi, NotebookLM, Chronicle.
Промт #1 — Структура презентации Claude / ChatGPT / Gemini
Ты – презентационный стратег McKinsey. Мой материал: [вставить текст или тему]. Правила – нарушать нельзя: 1. Слайд 1 – крючок: факт, который удивит, вопрос без очевидного ответа, или сцена, которую аудитория узнает. НЕ "Добро пожаловать". НЕ название темы 2. Заголовок = вывод. Тест: прочитай ТОЛЬКО заголовки – вся история понятна? 3. Для каждого слайда: – Заголовок-вывод (≤ 8 слов) – 2-3 факта. Числа > слова – Какой визуал: график / схема / иконка / крупный текст на пустом фоне – Одна фраза-мост к следующему слайду 4. Арка: Проблема (2-3 сл.) → Решение (3 сл.) → Доказательства (2 сл.) → Действие (1 сл.) 5. Последний слайд: одно действие + дедлайн. НЕ "Спасибо". НЕ "Вопросы?" Аудитория: [кто] Цель: [что СДЕЛАЮТ после – одно действие] Слайдов: [число]

  • Инфографика, презентации, mindmap, аудиоподкасты — всё в одном
  • Правки слайдов промтами прямо внутри: «Перепиши заголовки в выводы»
Промт #3 — NotebookLM Add Instructions
Роль: дизайнер презентаций McKinsey. Аудитория: [предприниматели / клиенты / инвесторы] Для каждого слайда: — Заголовок = вывод. Не "Продажи Q4", а "Продажи Q4 выросли на 47% год к году" — Максимум 3 тезиса, каждый ≤ 12 слов — Каждый тезис = факт из источника с цитатой Первый слайд: самый удивительный инсайт из материала. Тот, ради которого стоит смотреть дальше. Предпоследний: сравнение "было → стало" Последний: одно действие. Не "Спасибо" Формат: Detailed Deck Стиль: минималистичный, много воздуха Язык: русский

Gamma

Палочка-выручалочка — презентация за 3 минуты

gamma.app
Суперсила: скорость. Когда вызвали на ковёр и слайды нужны через 5 минут
  • Бесплатный тариф — 300 кредитов, хватает на 30+ презентаций
  • Брендбук: загрузите цвета и логотип — все презентации в вашем стиле
  • Экспорт в PDF, PPTX
  • После генерации: Edit with AI → «Каждый заголовок — это тема или вывод? Если тема — перепиши в вывод»
Промт #4 — Gamma gamma.app
Тема: [ваша тема] Дек для [аудитория], цель – [одно действие]. Жёсткие правила: 1. Слайд 1: вопрос, на который аудитория НЕ знает ответ. Или число, которое удивит 2. Тест: прочитай ТОЛЬКО заголовки всех слайдов подряд – история понятна целиком? Если нет – переписать заголовки 3. Буллит ≤ 10 слов. На слайде ≤ 3 буллита 4. Ноль стоковых картинок. Нужен визуал? График, диаграмма или иконка 5. Запрещённые фразы: "в современном мире", "стоит отметить", "важно понимать" 6. Последний слайд: действие + дедлайн Карточек: 10 · Язык: русский

Kimi

Бесплатный исследователь — сам ищет данные

kimi.ai
Суперсила: deep research. Адаптивные слайды с актуальными данными. Бесплатно
  • Два режима: Adaptive (глубокий ресёрч, 30-60 мин) и Visual (быстро, 5 мин)
  • NanoBanano Pro — генерация фотографий прямо внутри
  • Конкурирует с Claude по качеству текстов, растёт быстро
  • Бесплатный, без ограничений на количество презентаций
Промт #5 — Kimi kimi.ai · Adaptive
Создай презентацию с исследованием. Тема: [тема] · Режим: Adaptive ПРАВИЛА РЕСЁРЧА: — Только данные 2024-2026. Старее – игнорировать — Каждая цифра – со ссылкой на источник — Найди один факт, который ПРОТИВОРЕЧИТ общепринятому мнению по теме СТРУКТУРА: Слайд 1: "Все думают X. Данные говорят Y" Слайды 2-4: реальная картина с актуальными данными Слайд 5: таблица сравнения – кто делает хорошо, кто отстаёт Слайды 6-8: что это значит для [аудитория]. Практические выводы, не теория Слайды 9-10: 3 действия по убыванию импакта. #1 = максимальный ROI, делать первым Слайд 11: все ссылки на источники Слайдов: 11
Выбор режима Adaptive = глубокий ресёрч (30-60 мин, качество). Visual = быстро (5 мин, поверхностно). Для важных презентаций — всегда Adaptive.

Chronicle

chronicle.io
Суперсила: сторителлинг. Строит историю, не набор слайдов. Впечатляющий визуал
  • Стильные шаблоны, красивый визуал из коробки
  • Редактируется: шрифты, картинки, тексты — полный контроль
  • Экспорт в Google Slides
  • Фишка: вставьте URL вашего сайта — Chronicle сам построит питч из содержания
Промт #6 — Chronicle chronicle.io
Создай питч-дек, который рассказывает историю. Тема: [тема] Аудитория: [кто]. Они скептики. Нарративная арка: — Слайд 1: конкретный момент, который аудитория узнает. Не абстракция – сцена — Слайды 2-3: цена бездействия. Сколько времени/денег теряют в месяц — Слайд 4: что изменилось в мире, что делает новое решение возможным ИМЕННО СЕЙЧАС — Слайды 5-6: решение. До/после. Показывай, не рассказывай — Слайд 7: ОДНО доказательство. Одна метрика, одна цитата. Не три – одно, но неопровержимое — Слайды 8-9: как это работает. Максимум 3 шага — Слайд 10: одна просьба. Одно действие. Один дедлайн Тон: уверенный, как умный друг

Подпиши точку перелома, где всё изменилось — Слайд 4: таблица сравнения. 4+ колонки. Цвет: зелёный = лидер, красный = отстающий — Слайд 6: структура. Пирог или treemap. Проверьте себя

6 вопросов по материалу воркшопа. Кликните, чтобы увидеть ответ

01 Какой инструмент лучше всего подходит для быстрой презентации за 5 минут?
Ответ
Gamma. Суперсила Gamma — скорость. 3 минуты до готового дека. Бесплатный тариф с 300 кредитами, экспорт в PDF и PPTX. Когда вызвали на ковёр — это ваш инструмент.
02 Что должно быть в заголовке каждого слайда?
Ответ
Вывод, а не тема. Не «Продажи Q4», а «Продажи Q4 выросли на 47% год к году». Тест: прочитайте только заголовки подряд — вся история должна быть понятна без текста слайдов.
03 Какой инструмент лучше всего работает с данными и графиками?
Ответ
GenSpark. Правило для GenSpark: 70% графиков, 30% текста. Тренды, таблицы сравнения, пироговые диаграммы, treemap — всё из коробки. Идеален для превращения Excel в красивую презентацию.
04 Что такое NanoBanano Pro?
Ответ
Генерация фотографий в Kimi. Встроенная функция для создания фотоконтента прямо внутри Kimi. Не нужно переключаться в отдельный генератор изображений.
05 Какой инструмент работает как агент — ставишь задачу и уходишь?
Ответ
Manus. Полная автоматизация: сам ищет данные, сам дизайнит, экспортирует в .pptx. Ставите задачу, уходите от компьютера, возвращаетесь — презентация готова. Бонус: загрузите свой PPT — скопирует ваш стиль.
06 Какие три шага превращают чужой визуальный стиль в ваш?
Ответ
Скриншот → Gemini → ТЗ. Шаг 1: скриншот того, что нравится. Шаг 2: промт «Опиши визуальный стиль» + картинка в Gemini. Шаг 3: получаете описание стиля — вставляете в любой AI-инструмент как ТЗ.
