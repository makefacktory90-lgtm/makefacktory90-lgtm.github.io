const frame = document.querySelector('.ig-frame'); frame.style.cssText = 'width:420px;height:525px;max-width:none;border-radius:0;box-shadow:none;overflow:hidden;margin:0;'; const viewport = document.querySelector('.carousel-viewport'); viewport.style.cssText = 'width:420px;height:525px;aspect-ratio:unset;overflow:hidden;cursor:default;'; document.body.style.cssText = 'padding:0;margin:0;display:block;overflow:hidden;'; }""") await page.wait_for_timeout(500) for i in range(TOTAL_SLIDES): await page.evaluate("""(idx) => { const track = document.querySelector('.carousel-track'); track.style.transition = 'none'; track.style.transform = 'translateX(' + (-idx * 420) + 'px)'; }""", i) await page.wait_for_timeout(400) await page.screenshot( path=str(OUTPUT_DIR / f"slide_{i+1}.png"), clip={"x": 0, "y": 0, "width": VIEW_W, "height": VIEW_H} ) print(f"Exported slide {i+1}/{TOTAL_SLIDES}") await browser.close() asyncio.run(export_slides()) ``` ### Критичные правила экспорта - **Python для HTML** — НИКОГДА shell scripts (переменные ломают HTML) - **Base64 для картинок** — всё встроено в HTML - **Viewport 420×525** — НЕ 1080×1350 (device_scale_factor масштабирует) - **Ждать шрифты** — `wait_for_timeout(3000)` после загрузки - **Прятать IG frame** — `.ig-header,.ig-dots,.ig-actions,.ig-caption` --- # ═══════════════════════════════════════════ # BATCH MODE: 5 каруселей за раз # ═══════════════════════════════════════════ Если пользователь хочет сделать несколько каруселей: 1. Пошагово

Как собрать систему

01

Откройте свой проект в Claude

Тот же проект, что вы создали на Уроке 2. Если нет — создайте новый: claude.ai → Projects → New Project.

02

Замените промпт на новый

В настройках проекта (Project Instructions) замените старый промпт из Урока 2 на новый промпт-систему из этого урока.

03

Проверьте файлы фундамента

В проекте должны лежать 4 файла из Урока 1: голос, боли, продукт, стоп-лист. Если их нет — добавьте.

04

Скажите Claude тему

«Сделай карусель про...» — и система поведёт: research → хук → формула → сценарий → критик → вёрстка → слайды.

05

Или попросите батч

«Придумай 5 тем на неделю» — Claude выдаст темы с хуками и формулами. Выберите и прогоните через систему.

Бонус — следующий уровень

Как превратить промпт в скилл

Сейчас вы вставляете промпт в начало каждого чата. Это работает, но есть способ лучше: сделать из него скилл — промпт, который живёт у Claude постоянно. Просто пишете «сделай карусель» — и Claude сам подтягивает все инструкции. Без копипаста, без потерь.

Что понадобится
Аккаунт Claude Pro, Team или Enterprise
Плюс ваш готовый промпт из этого урока и 15–20 минут
01

Подготовьте фактуру

Убедитесь, что в промпте есть всё из Урока 1 (голос, боли, продукт, стоп-лист) и Урока 2 (цвет, стиль, шрифты, handle). Это основа скилла.

02

Попросите Claude сделать скилл за вас

Откройте новый чат, вставьте промпт и напишите:

Преврати этот промпт в скилл для Claude. Создай файл SKILL.md с правильной шапкой: - name: carousel-maker - description: подробное описание когда срабатывать — включи прямые запросы («сделай карусель», «нужны слайды») и косвенные («подготовь визуал для поста», «контент-план для Instagram») Разбей инструкции: - Основной SKILL.md — до 500 строк (Phase 0, 1, 2, 3 + ссылки на reference-файлы) - references/formulas.md — все 16 формул с правилами - references/design.md — архитектура слайдов, CSS-компоненты, палитра, шрифты Упакуй всё в .skill файл.
03

Скачайте .skill файл

Claude создаст все файлы и упакует их. Скачайте архив.

04

Установите скилл

Настройки Claude → Skills → загрузите .skill файл. Готово.

05

Проверьте

Откройте новый чат и напишите: «Сделай карусель про выгорание» или «Придумай 5 тем на неделю». Если Claude подхватил — скилл работает.

→ Слишком длинный SKILL.md — если файл больше 500 строк, Claude может «забыть» инструкции. Выносите детали в references/
→ Слабый description — если скилл не срабатывает, добавьте слово-триггер в description
→ Объясняйте почему — вместо «НИКОГДА не пиши слово уникальный» лучше: «Избегай слова уникальный — оно звучит как корпоративный шаблон и снижает доверие»
