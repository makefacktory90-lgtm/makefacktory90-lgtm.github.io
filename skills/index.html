Что такое «скилл»

Скилл — это markdown-файл, который можно положить в Claude Code (или Claude Projects) как инструкцию. Claude читает, выполняет. Никакого кода, только текст + структура.

Как использовать — 3 уровня

L1 — Ручной (любой чат с AI)

Claude.ai / ChatGPT / Gemini — работают как есть. Открой скилл, найди раздел «Процесс», скопируй промпты вручную. Время: 2-4 часа.

L2 — Облегчённый (Claude Code)

Нужен Claude Code + MCP для веб-поиска (10 минут настройки). Скачай скилл → положи в ~/.claude/skills/ → запусти. Время: ~25 минут.

L3 — Полный (Claude Code + PAL MCP + OpenRouter)

Всё как у меня. 4 модели параллельно, синтез за 15 минут. Настройка один раз ~20 минут + $5 на OpenRouter хватит на 2-3 месяца. Подробности внутри скилла.
