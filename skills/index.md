# ira-and-ai — публичные скиллы

Публичная коллекция AI-скиллов, промптов и инструментов от [Иры Буян](https://t.me/ira_and_ai).

Здесь — то, что я использую сама и отдаю людям. Бесплатно, с лицензией MIT.

## Что такое "скилл"

Скилл — это markdown-файл, который можно положить в Claude Code (или Claude Projects) как инструкцию. Claude читает, выполняет. Никакого кода, только текст + структура.

## Скиллы

### 🔍 Исследование ЦА за 15 минут
[`skills/SKILL-research-icp-v1.md`](skills/SKILL-research-icp-v1.md)

Портрет целевой аудитории через 3 нейросети параллельно + локальные источники. В одном окне Claude Code. Вдохновлено [статьёй на Хабре](https://habr.com/ru/articles/1018360/) — там то же самое делают руками за 2 часа, здесь — за 15 минут, без ручной пересадки между вкладками.

**Для кого:** маркетологи, продуктологи, эксперты, запускающие новый продукт или курс.

## Как использовать — 3 уровня

Скилл работает на разных уровнях инфраструктуры. Выбирай свой:

### L1 — Ручной (любой чат с AI, без настройки)

Claude.ai / ChatGPT / Gemini — работают как есть.

1. Открой скилл, найди раздел "Процесс"
2. Скопируй промпт Шага 1 (метапромптинг) → вставь в любую модель
3. Получи оптимизированный промпт
4. Скопируй его в **3 разных окна**: Claude, ChatGPT, Gemini
5. Сравни ответы вручную, собери синтез в ещё одном окне

**Время:** 2-4 часа (как в [статье на Хабре](https://habr.com/ru/articles/1018360/), которая нас вдохновила).
**Плюс:** работает без настроек.
**Минус:** устаёшь, легко потерять контекст.

### L2 — Облегчённый (Claude Code)

Нужен [Claude Code](https://www.anthropic.com/claude-code) + MCP для веб-поиска (10 минут настройки).

1. Скачай скилл в `~/.claude/skills/` или в папку проекта
2. Запусти Claude Code в этой папке
3. Команда: *"запусти SKILL-research-icp для продукта X"*
4. Claude через субагенты соберёт данные, сделает синтез

**Время:** ~25 минут.
**Плюс:** одно окно, субагенты, локальные файлы читаются напрямую.
**Минус:** только один "голос" (Claude), без Метода Франкенштейна.

### L3 — Полный (Claude Code + PAL MCP + OpenRouter)

Всё как у меня. Настройка один раз ~20 минут + $5 на OpenRouter хватит на 2-3 месяца.

1. Получи API-ключ на [openrouter.ai/keys](https://openrouter.ai/keys)
2. Поставь [PAL MCP](https://github.com/BeehiveInnovations/pal-mcp-server) (бывший Zen): `git clone` + `./run-server.sh`
3. В `.env` PAL впиши `OPENROUTER_API_KEY=...`
4. Зарегистрируй PAL в Claude Code: `claude mcp add pal -s user ...`
5. Перезапусти Claude Code
6. Запусти скилл — 4 модели параллельно, синтез за 15 минут

**Время:** ~15 минут на каждое исследование ЦА.
**Плюс:** полный Метод Франкенштейна — Claude + GPT + Gemini + DeepSeek голоса в одном отчёте. Локальные источники. Один клик.
**Минус:** требует начальной настройки.

---

**Моя рекомендация:** начни с L1 на одной своей задаче — проверь ценность метода. Понравится — настраивай L3.

## Что требуется

Для большинства скиллов:
- Claude Pro / Max / Team (для Claude Code)
- MCP для веб-поиска (опционально)

Для скилла "Исследование ЦА" — желательно OpenAI MCP + Gemini MCP для полного Метода Франкенштейна. Без них работает, но чуть беднее.
