Зачем вам Claude Code, если вы не собирались кодить

Я давно хотела написать это письмо. Всё тянула, потому что боялась, что прозвучит как реклама Claude Code.

Сегодня поняла, что по-другому не объяснить. Просто покажу на живой задаче.

Один из подписчиков прислал задачку: федеральный вуз в Калининграде, задача, переманивать научных сотрудников из Москвы, делать предложение конкурентоспособным по всей РФ.

Три пути решить одну и ту же задачу: руками, через нейросети руками, и через мой скилл.

01 / Маркетолог без AI (руками)

Как это делали ещё вчера:

  • Составить бриф и анкету, 2-3 часа
  • 10-15 глубинных интервью, 2-3 недели
  • 1-2 фокус-группы, ещё неделя
  • Расшифровать, свести в Excel, написать отчёт, 3-5 дней

Итого: 2-3 недели, помощники, логистика, голова дымится.

У 99% маркетологов столько времени нет. Поэтому это никто не делает. Портреты ЦА пишутся на коленке за 30 минут по воспоминаниям. Результат, слабый, угадательный.

02 / Маркетолог с нейросетями (Хабр-метод)

Недавно на Хабре вышла статья «Исследование ЦА за 2 часа через AI», красивый метод, где тот же запрос прогоняется через три нейросети параллельно, руками. Любопытным, сюда.

4 окна. Ручная пересадка. 2-4 часа с настройкой.

Радикально быстрее, чем руками-без-AI. Но каждый раз всё заново: четыре вкладки, своя подписка на каждую модель, конспектирование в Google Docs и общая усталость к концу дня.

Я прошла этот метод один раз и задумалась: а почему нельзя всё в одном окне?

03 / Маркетолог со скиллом (мой способ)

Я вдохновилась методом Хабра и написала свой скилл для Claude Code.

Скилл, это markdown-файл с инструкцией. Claude Code его читает и выполняет. Никакого программирования. Никаких четырёх окон.

Клод читает инструкцию из скилла, сам опрашивает несколько AI-моделей, сам собирает локальные источники (если они есть), сам пишет финальный отчёт. Я сижу рядом и пью кофе.

Одна команда. Одно окно. На задаче про вуз справился примерно за час.

Живой результат на задаче про вуз в Калининграде

Выложила отдельной страницей, без регистрации и СМС:

→ iraai.ru/bfu-demo

Там три портрета ЦА (включая «молодого постдока с женой-архитектором и ребёнком у тёщи»), пять углов позиционирования, контринтуитивные стратегии вроде «перестаньте продавать им работу, продавайте свободу и гранты под их именем».

Там же мои комментарии к каждому блоку, что проверила руками, с чем не согласна, где модель попала в точку.

05 / Зачем вам Claude Code, если вы не собирались кодить

Вот главная идея этого письма.

Claude Code придумали разработчики для разработчиков. Но выяснилось, что он радикально сильнее для всех остальных, если научиться писать или использовать скиллы.

Скилл, это обычный markdown-файл. Никакого программирования. Никакого терминала в вашей ежедневной работе с ним.

Зачем это маркетологу, эксперту, автору:

1. AI работает ЗА вас, а не РЯДОМ с вами

В ChatGPT вы сидите и печатаете промпты. С Claude Code, запускаете один скилл, идёте за кофе, возвращаетесь к готовому отчёту.

2. AI помнит контекст между сессиями

В ChatGPT каждая новая сессия, чистый лист. В Claude Code есть файл about-me.md: кто вы, ваша аудитория, ваш голос. Читает и применяет. Не нужно каждый раз объяснять заново.

3. AI работает с вашими файлами

Telegram-экспорт, опросы, интервью, ваши статьи, Claude Code читает напрямую, с вашего компьютера. Не нужно ничего загружать в облака чужих сервисов.

🍸 Короче: Claude Code, это не «ещё один чат». Это рабочее пространство, где вы ставите задачу и идёте заниматься своими делами. Я как человек, который искренне любит, когда за меня работают, просто не могу на это смотреть без нежности.

06 / Забрать скилл

Скилл, это просто markdown-файл с инструкцией. Работает двумя способами:

Руками. Открываете любую нейросеть (Claude, ChatGPT, Gemini), копируете промпты из скилла по одному, получаете результат. Как в статье с Хабра. Дольше, но работает.

Через Claude Code. Кладёте файл в папку, клод читает инструкцию и делает всё сам, пока вы занимаетесь своими делами. Быстрее, повторяемо, один клик.

Как именно подключать, написано внутри самого скилла на GitHub. Там же ссылки на всё нужное.

→ github.com/tiara111/ira-and-ai

Лицензия MIT. Берите, используйте, адаптируйте под свою нишу.

Если улучшите, пришлите Pull Request. Добавлю в v2 с вашим именем.

07 / Одно правило

Цифры, которые выдаст модель, проверяйте сами перед тем как вставить в лендинг, продающий текст или инвестиционный документ.

Скилл возвращает углы зрения, не готовые решения. Финальное решение о стратегии, цене, приоритетах, ваше. Скилл, расширенный мозг, не замена вашему.

Это правило встроено в сам скилл. Прочитайте перед использованием.
