Что изменилось в v4

Новая философия: Opinionated Bot

v3: бот предлагает 3 варианта, пользователь выбирает, критик проверяет, пользователь утверждает. 5-7 решений на одну карусель.

v4: бот приносит одно лучшее решение. Как хороший дизайнер: не спрашивает "синий или красный?", а приносит готовый макет. 2 касания вместо 7.

Retention Engine

v3: бот реактивный. Пользователь приходит сам, когда вспомнит.

v4: бот проактивный. Weekly push по вторникам. Обратная связь через 48ч. Streak. Привычка = возвратность = деньги.

Тарифы

v3: Free = 3 карусели/месяц (хватает casual-пользователю навсегда).

v4: Free = 1 карусель всего. Хватит увидеть магию. Не хватит пользоваться. Starter $9-15 — main tier.

Под капот

Критик-агент — невидим пользователю (был отдельный шаг). 3 варианта текста — убраны (был паралич выбора). Формула — подбирается автоматически (была ручная).

Что делает бот

Не "делает карусели". Приводит клиентов. Пользователь приходит в Telegram. Через 5 минут уходит с готовыми карусельными картинками, написанными по проверенным маркетинговым формулам, адаптированными под его голос, нишу и аудиторию. Карусели, которые продают, а не просто красиво выглядят.

Оффер: Маркетолог + копирайтер + трендвотчер + AI-дизайнер в одном Telegram-боте. 16 маркетинговых формул. Тренды твоей ниши. Твой голос. Готовые картинки. Клиенты из Instagram, а не лайки.

IP продукта

Бот без формул = ещё одна ChatGPT-обёртка. Формулы, промты, flow — вот что делает его ценным. Это интеллектуальная собственность Иры: 16 маркетинговых формул с правилами, промты с few-shot примерами, логика подбора формулы под цель, критик-агент, платформенный интеллект. Денис строит платформу. Ира вкладывает мозги.

4 встроенные возможности роста

Возможность 1

Weekly Push = привычка = возвратность

Бот сам приходит по вторникам: "3 темы в твоей нише на этой неделе. Какая?" Пользователь не должен помнить про бот — бот помнит про него. Привычка = регулярное использование = оплата подписки. Без push — пользователь забудет через неделю.

Возможность 2

Бот = лид-машина для дорогих услуг

Бот знает голос, стиль, бизнес клиента. Идеальная точка для апселла: "Хотите лендинг в этом же стиле?" — переход на услугу 50-100K. Upsell заложен в flow: после 10-й карусели бот предлагает "Полную упаковку бренда". Не навязчиво. Как рекомендация от того, кто уже знает твой бизнес.

Возможность 3

Видимые формулы = выше ценность, ниже отток

Каждую карусель бот выдаёт с пометкой: "Формула: PAS (Боль → Усиление → Решение)". Клиент учится, пока пользуется. Perceived value растёт. Отток падает — люди чувствуют, что растут, а не просто жмут кнопку. Через месяц: "Я знаю 5 формул продающего контента" — это то, за что платят.

Возможность 4

Автоматическое социальное доказательство

Каждый клиент бота делает карусели для своего бизнеса. В flow заложено: после 10-й карусели бот спрашивает "Как результаты? Хочешь поделиться?". Автоматический поток кейсов. Без участия Иры. На каждой карусели мелким: "Сделано в @carousel_bot" — каждая публикация = бесплатная реклама.

Два режима визуала

Все картинки генерируются AI. Разница в уровне: с текстом на русском или без.

STANDARD: AI-картинки без текста

  • Nano Banana генерирует фон/композицию
  • Текст НЕ на картинке (отдельно)
  • Дешевле в генерации
  • Красивый визуал, уникальный каждый раз
  • Пользователь накладывает текст сам (или в подписи)
  • Хватает для старта

PRO: AI-картинки с русским текстом

  • Текст прямо на картинке (русский)
  • Готовые слайды — скачал и постишь
  • Промт-инжиниринг под капотом
  • Правила генерации вшиты в бот
  • Уровень, который руками не сделать
  • Дороже, но ценность очевидна
Upsell логика: Standard показывает красивый фон + текст отдельно. Тут же превью: "А вот так выглядит с текстом на картинке (Pro)". Разница моментальная. Конверсия в Pro будет высокой, потому что человек уже увидел свой текст на красивой картинке и хочет именно так.

Пользовательский путь v4: Opinionated Bot

Принцип: Бот приносит одно лучшее решение. Не спрашивает "какой из трёх вариантов?". Не показывает критика. Автоматически подбирает формулу. Пользователь выбирает тему — получает готовую карусель. Хочет поправить — может. Но по умолчанию — одно касание и готово.

Фаза 2

Профиль пользователя

Сохраняется навсегда. Обогащается с каждым использованием. При повторном визите — сразу к теме.

Структура профиля
{ "user_id": 123456, "name": "Маша", "niche": "нутрициология", "expertise": "интуитивное питание, РПП", "audience": "женщины 28-40, устали от диет", "goal": "продажи", "tone_of_voice": "на ты, тёплый но прямой, с юмором", "visual_style": "minimal_light", "brand_colors": {"primary": "#1a1a1a", "accent": "#e85d3a"}, "plan": "starter", "carousels_made": 0, "streak": 0, "formulas_used": [], "last_carousel_date": null, "created_at": "2026-02-27" }
Lock-in: Чем больше пользуется, тем точнее бот знает стиль. Уйти к конкуренту = заново объяснять кто ты. Профиль = привязка. Плюс: бот ротирует формулы (не повторяет PAS три раза подряд), и для этого нужна история.
Фаза 3

Тема: push или своя

Путь A: Weekly Push (основной). Вторник, 10:00. Бот сам приходит с 3 темами. Пользователь тапает — карусель генерируется.

Промт: weekly push (3 темы)

Под капотом

Авто-формула + генерация + невидимый критик

Пользователь НЕ видит этот шаг. Бот делает всё сам за 20-40 секунд.

  1. Авто-формула. Бот подбирает формулу по: (1) цель пользователя, (2) тип темы, (3) ротация — не повторять последние 3 формулы. Таблица маппинга ниже.
  2. Генерация текста. Один вариант (не три). Промт включает формулу + профиль + платформенные правила. Few-shot пример вшит.
  3. Невидимый критик. Тот же промт критика из v3, но результат НЕ показывается пользователю. Если оценка < 7/10 — автоматический перегенер. Если ≥ 7 — выдаётся как готовое.
  4. Финализация. Текст + подпись + формула-тег. Одно сообщение пользователю.

Визуал

Генерация картинок

Два режима по тарифу. Пользователь не выбирает — зависит от подписки.

Режим STANDARD (Starter): AI-картинки без текста

  1. Промт. Бот генерирует промт для Nano Banana: фон, композиция, настроение. Без текста на картинке.
  2. Генерация. Nano Banana → уникальное AI-изображение для каждого слайда.
  3. Доставка. Альбом AI-картинок + текст отдельно (для наложения в Canva или как подпись).
  4. Upsell. Тут же превью Pro-версии: "А вот так выглядит с текстом на картинке."

Режим PRO: AI-картинки с русским текстом

  1. Промт-инжиниринг. Бот берёт финальный текст каждого слайда и генерирует промт: текст + фон + композиция + типографика.
  2. Генерация. Модель генерирует картинку с русским текстом прямо на изображении.
  3. Consistency. Первый слайд → стилевой якорь → остальные слайды в том же стиле.
  4. Доставка. Альбом готовых слайдов. Скачал → постишь. Подпись прилагается.

Retention Engine

Три механизма, которые превращают разовое использование в привычку.

Retention 1

Weekly Push: бот приходит сам

Вторник, 10:00. Бот отправляет 3 темы. Пользователь тапает — карусель за 30 секунд.

Четверг, 10:00. Если не зашёл: "Тема #1 набирает обороты. Карусель за 2 минуты?"

Пятница, 18:00. Если не зашёл: тишина. Не спамим. Одно напоминание в неделю, не больше.

Только платные тарифы. Weekly push — фича Starter и Pro. Free-пользователи не получают. Это дополнительный стимул платить: push = привычка = рост аккаунта. Без push = забудешь.
Retention 2

Обратная связь: 48 часов после карусели

Через 48 часов после генерации:

БОТ: "Маша, как карусель про читинг? Сколько сохранений? Комментарии? [Круто, 50+ сохранений] [Нормально, 10-50] [Плохо, меньше 10] [Не постила ещё]"

Зачем: (1) Обогащает профиль — бот узнаёт, какие формулы работают для этого пользователя. (2) Возвращает в бот. (3) Если "Круто" — предложить поделиться кейсом. Если "Плохо" — предложить другую формулу.

Без API Instagram. Просто спрашиваем. Данные субъективные, но достаточные для обучения бота и retention. API Instagram для аналитики — v3, если будет нужно.
Retention 3

Streak + прогресс

Тонкая геймификация. Без наград — просто факт.

После каждой карусели: "Карусель #7. Формула: Контраст. Ты использовала 5 из 16 формул. Серия: 3 недели подряд." После 4 недель подряд: "Маша, 4 недели подряд! Эксперты, которые постят каждую неделю, растут в 3 раза быстрее. Следующая формула, которую стоит попробовать: Скользкая горка (экспертность)." После 10 каруселей: "10 каруселей! Ты освоила 6 формул. Как результаты? Хочешь поделиться?" → автоматический сбор кейсов

Библиотека формул (16 штук)

Ядро продукта. Каждая формула: структура слайдов, правила переходов, пример, типичные ошибки. IP Иры.

Продажи

1. PAS — Problem → Agitate → Solution

Классика прямых продаж. Давление через боль, усиление, спасение.

Структура (10 слайдов):

1HOOKПровокационное утверждение о проблеме. Max 8 слов. "Ты постишь каждый день. Клиентов — ноль."
2PROBLEMКонкретная боль. Не абстрактная — с деталями. "Ты пишешь посты. Лайки ставят мама и подруга."
3PROBLEM+Углубление: почему это больно. "А коллега в той же нише — 200 сохранений на каждый пост."
4AGITATEУсиление: последствия бездействия. "Через полгода ты устанешь и бросишь."
5AGITATE+Эмоциональный пик: страх/стыд/злость. "А деньги на рекламу, курсы, фотосессии — уже потрачены."
6PIVOTПоворот: "Но есть кое-что, что работает." Без подробностей — интрига.
7SOLUTIONРешение: что именно. Конкретно.
8PROOFДоказательство: цифра, кейс, факт.
9HOWКак получить решение. Просто.
10CTA"Сохрани и попробуй. Ссылка в шапке."

Правила: Слайды 2-5 нарастают по интенсивности. Слайд 6 — резкий слом ритма. Нельзя давать решение раньше слайда 7.

Ошибки: Слишком мягкая боль. Решение на слайде 3 (рано). CTA "подпишись" вместо "сохрани".

Продажи

2. BAB — Before → After → Bridge

Трансформация. Мир "до" и "после", потом мост.

1HOOKКонтраст в одной фразе. "3 месяца назад я плакала в туалете офиса. Сейчас у меня 40 клиентов."
2-3BEFOREПодробное "до": детали, эмоции, цифры.
4TURNINGМомент перелома.
5-6AFTERПодробное "после": те же метрики, другие цифры.
7-8BRIDGEЧто конкретно изменилось. Максимум 3 шага.
9TAKEAWAYГлавный вывод.
10CTA"Сохрани, если узнала себя в 'до'."

Правила: "До" болезненно конкретным. "После" реалистичный. Мост — максимум 3 шага.

Ошибки: "До" абстрактно. "После" нереалистично. Мост = реклама.

Запуски

3. AIDA — Attention → Interest → Desire → Action

Воронка внимания. Для запусков и анонсов.

1ATTENTIONШок, неожиданность, цифра.
2-3INTERESTПочему важно именно тебе. Факты.
4-5DESIREКартинка: как будет. Сенсорные детали.
6-7PROOFСоцдоказательство.
8OFFERЧто предлагается.
9URGENCYПочему сейчас.
10ACTIONОдно действие.

Правила: Attention ≠ кликбейт. Desire через ощущения. Action — одно действие, не три.

Экспертность

4. Скользкая горка (Шугерман)

Каждый слайд тянет к следующему. Нет слайда, на котором можно остановиться.

1HOOKОбещание + интрига.
2SETUPКонтекст. Заканчивается обрывом.
3-7SLIDEКаждый обрывается на полумысли.
8-9PAYOFFКульминация: раскрытие.
10CTA"Сохрани — перечитаешь."

Правила: КАЖДЫЙ слайд — незавершённая мысль. Обязательно. Техники: многоточие, "но...", вопрос.

Охваты

5. Open Loop — Незакрытая петля

Обещание на первом слайде. Отдача на последнем. Убийца сохранений.

1PROMISEКонкретное обещание.
2-4CONTEXTПочему важно. НЕ раскрывает обещание.
5-7BUILDПриближение к ответу.
8-9REVEALРаскрытие. Конкретно.
10CTA"Сохрани — эту формулу ты будешь использовать."

Правила: Обещание ДОЛЖНО быть выполнено. Каждый слайд между обещанием и раскрытием даёт самостоятельную ценность.

Охваты

6. Контраст — Миф vs Реальность

Попарные противопоставления. Провокация + разрушение мифов = вирусность.

1HOOK"5 вещей, которые тебе врали про [тему]."
2-3ПАРА 1Миф → Реальность.
4-5ПАРА 2Миф → Реальность. Острее.
6-7ПАРА 3Самый сильный контраст.
8-9ВЫВОДЧто делать.
10CTA"Перешли подруге, которая верит в [миф]."

Правила: Пары нарастают по остроте. Реальность — с доказательством.

Подписки

7. Сторителлинг — Герой → Проблема → Путь → Результат

Личная история или кейс. Эмоция = подписки.

1IN MEDIAS RESНачать с середины.
2BACKSTORYКто герой. Контекст.
3-4CHALLENGEПроблема с деталями.
5ATTEMPTЧто не работало.
6TURNING POINTЧто изменилось.
7-8ACTIONЧто конкретно сделала.
9RESULTЦифры. До и после.
10CTA"Если узнала себя — сохрани."

Правила: In medias res обязательно. Детали: имена, цифры, места. Один поворотный момент.

Экспертность

8. How-To / Listicle

Пошаговый гайд или список. Чистая ценность.

1HOOKЧёткое обещание с цифрой.
2-9ШАГИПо одному на слайд. Заголовок жирным + пояснение.
10CTA"Сохрани как чеклист."

Правила: Каждый шаг — одно действие. Конкретные (можно выполнить за 15 минут).

Запуски

9. FAB — Feature → Advantage → Benefit

Продуктовая формула. Свойство → преимущество → польза.

1HOOKРезультат продукта.
2-3FAB 1Фича → Преимущество → Польза.
4-5FAB 2Другой аспект.
6-7FAB 3Killer feature.
8SOCIAL PROOFКто уже использует.
9OFFERЧто входит, сколько стоит.
10CTAОдно действие.

Правила: Benefit всегда про пользователя, не про продукт.

Охваты

10. Провокация — "Перестань делать X"

Самый вирусный формат.

1УДАРКатегоричное утверждение.
2УТОЧНЕНИЕУсиливаем, добавляем контекст.
3-7АРГУМЕНТЫПо одному на слайд. Факты, нарастание.
8АЛЬТЕРНАТИВАЧто делать вместо.
9ИТОГРезюме.
10CTA"Согласна? Пиши в комментариях." (дискуссия = охваты).

Правила: Аргументы — факты, не мнения. Альтернатива обязательна.

Охваты

11. Cliff / Cliffhanger

Каждый слайд обрывается на полуслове.

1HOOKНачало фразы, которая обрывается.
2-8CHAINКаждый слайд продолжает предыдущий И обрывается.
9PAYOFFФинальное раскрытие.
10CTA"Сохрани, чтобы перечитать."

Правила: КАЖДЫЙ слайд обрывается. Без исключений.
